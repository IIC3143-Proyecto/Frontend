# Vtrina Frontend

Frontend en Next.js con App Router, Tailwind CSS v4 y shadcn/ui.

## Stack

- Next.js 16.2.4 + React 19.2.4
- TypeScript
- Tailwind CSS v4 (via PostCSS)
- shadcn/ui + Radix UI
- TanStack React Query
- Lucide Icons

## Comandos

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Desarrollo

```bash
npm install
npm run dev
```

Luego abre http://localhost:3000.

## Estructura

- src/app: rutas, layout y estilos globales
- src/components/ui: componentes UI
- src/lib/utils.ts: utilidades (helper `cn`)
